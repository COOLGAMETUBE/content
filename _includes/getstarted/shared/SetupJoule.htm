What you need

  • From Step 1: A PC running Windows 10
  • From Step 1: A prepared USB drive with the JouleAutoInstaller, ACPITABL.dat, and FFU
  • Joule Development Platform
  • 12V, 3A power adapter (power adapter must be Listed LPS or Class 2 output rated, 12Vdc, 3A minimum) with barrel jack for powering the expansion board. The barrel dimensions are 5.5mm outer diameter, and 2.1mm inner diameter. The barrel length is ~9-10mm. The plug is positive center.
  • Micro HDMI cable and monitor
  • POWERED USB 3.0 hub. The development platform has only one Type A USB port. The Type B and C ports should not be used unless directed.
  • USB keyboard and Mouse
  • Other cords and peripherals as documented by Intel.

Update the firmware and install Windows 10 IoT Core

  1. First, we must update the firmware on the Joule device. While you may have just purchased the device, Intel has made recent firmware updates to improve the functionality and support Windows. Follow the process on Intel's website to download and install the latest firmware for Joule.
  2. If you haven't already, through the USB hub, connect the USB drive you created in Step 1, a USB keyboard, and a USB mouse.
  3. After you've flashed the device, power it up. When you see the Intel logo splash screen, immediately press F2 to enter the firmware setup. If you miss the prompt, reboot and try again.
  4. Confirm the firmware has been updated. It should read GTPPA16A.X64.0143.B30.1608112014
  5. From the firmware setup screen, Select Boot Manager -> EFI USB Device. Press Enter. The system will restart.
  6. When the system reboots, the JouleAutoInstaller will run from the USB and flash the FFU to Joule eMMC.
  7. When promoted, press any key to continue. This will take you the the WinPE prompt.
  8. From the WinPE prompt, switch to the USB drive. Usually, this is "d:", but you may need to try other drive letters.
  9. Type: "copy ACPITABL.dat c:\windows\system32".
  11. If asked, orerwrite the existing ACPITABL.dat file. This will copy the ACPITABL.dat file from your USB drive to your Windows 10 IoT Core file system on the eMMC.
  12. From the WinPE prompt, type: "wpeutil reboot"
  13. Press F2 to enter the firmware again.
  14. Go to Device Manager -> System Setup -> Boot and set "OS Selection" to Windows.
  15. Press F4 to save.
  16. Press ESC to go back one screen. Select Commit changes and exit.
  17. The system will boot to Windows 10 IoT Core, start the configuration process, and load the default app. You may remove the USB flash drive. You will no longer need it.
